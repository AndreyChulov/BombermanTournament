Игра гонки.

Игрок должен приехать к финишу первым.

На пути может попасться трамплин или бонус
Игрок управляет перемещением гонки по полосам и скоростью машины
бонус может дать ускорение (нитро)
бонус может дать замедление (камни)
бонус может дать случайное перемещение по полосам (масло)
выехать за границы трассы нельзя
